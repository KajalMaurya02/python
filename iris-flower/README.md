# 🌸 Iris Flower Classification

Basic classification project to classify Iris flowers into species using petal/sepal measurements.

## 🔧 Tools & Technologies
- Python
- Scikit-learn, Seaborn

## 📌 Highlights
- Clean dataset
- KNN and SVM classifiers
- Accuracy comparison with plots
